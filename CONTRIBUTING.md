# Contribuer à Need For Crous

Merci de l'intérêt que vous portez à ce projet ! Nous voulons rendre la contribution aussi simple et transparente que possible, que ce soit pour :

- Signaler un bug
- Discuter de l'état actuel du code
- Soumettre un correctif
- Proposer de nouvelles fonctionnalités

## Nous développons sur GitHub

Nous utilisons GitHub pour héberger le code, pour suivre les problèmes (issues) et les demandes de fonctionnalités, ainsi que pour accepter les pull requests.

## Signaler des bugs via l'outil de suivi GitHub

Nous utilisons les "Issues" GitHub pour suivre les bugs publics. Signalez un bug en [ouvrant une nouvelle issue](https://github.com/kira-mari/NeedForCrous/issues/new) ; c'est aussi simple que ça !

## Rédiger des rapports de bugs détaillés

**Un bon rapport de bug** doit contenir :

- Un résumé rapide et/ou le contexte
- Les étapes pour reproduire le problème
  - Soyez précis !
  - Donnez un exemple de code si possible.
- Ce que vous attendiez comme résultat
- Ce qui s'est réellement passé
- Des notes (pourquoi vous pensez que cela arrive, ce que vous avez essayé et qui n'a pas fonctionné)
